Comment partager mes contenus?

La licence CC BY-NC-SA 4.0 est bien plus permissive que la licence Copyright (appliquée par défaut à toute création), mais elle ne confère cependant pas tous les droits. Pour partager un contenu de mon site ou de ma page Instagram, vous devez impérativement:

  1. Inclure le nom d'auteur: A. Sweeney;
  2. Partager les contenus sans objectif commercial;
  3. Inclure cette même licence même s'il s'agit d'un travail effectué sur la base de ces contenus.
Aussi, veuillez s'il vous plaît inclure un hyperlien vers mon site ou ma page Instagram dans mon nom d'auteur ou l'indiquer de manière visible.
Par exemple, il est possible de traduire les contenus et de les diffuser sur les réseaux sociaux. Il est aussi possible de les inclure dans une oeuvre (tant que cette oeuvre n'est pas sur le marché). Il serait en revanche défendu de les vendre sous quelque forme que ce soit (ex.: livre, PDF, epub, livre audio, etc.) ou de vendre un travail issu de ces contenus (ex.: une traduction). Pour plus d'informations, veuillez consulter l'hyperlien inclus dans la licence.
